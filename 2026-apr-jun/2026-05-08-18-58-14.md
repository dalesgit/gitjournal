Wi-Fi captive portal pages load by hijacking an insecure HTTP (as opposed to secure HTTPS) page load. If you've only tried to load pages that you've previously loaded securely, or if you're running a Chrome extension like "HTTPS Everywhere" that forces everything to go over secure HTTPS if it can, then you might need to try to make an insecure HTTP connection one way or another.
